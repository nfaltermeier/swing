Swing. A platforming game. Navigate your blue square to the green square.

## Controls ##
* Movement - WASD
* Jump - Space (Longer press means higher jump)
* Interact with menu - Mouse
* Pause - Escape

Technical about
Made with monogame for DirectX using .NET Core. Uses Aether Physics2d.

Art made Paint.NET, audio made using BFXR. See ASSETS.TXT for credits for assets.
